# calculation

일반적으로 속력은 가속도의 적분입니다.

$V(s)=\int_{0}^{s}A(s)$

RTOS에서는 실시간 실행 (주기적 실행)을 보장하므로, 다음과 같이 생각할 수 있습니다.

$V(s)=\sum_{0}^{s}kA(s)$

이때, 상수 k는 실행 시간 간격에 대한 상수로, task 실행 주기에 따라 수치를 보정해주는 역할을 합니다.
